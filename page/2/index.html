Xcode插件

Xcode插件管理神器 —— Alcatraz

说明

Alcatraz是一个管理Xcode开源包的,你可以用它查找安装你想要的插件，模板以及配色方案，而你无需手动克隆他们，它像Xcode的一部分。

安装

curl -fsSL https://raw.githubusercontent.com/supermarin/Alcatraz/master/Scripts/install.sh | sh

Alcatraz只支持OSX 10.9+ Xcode5+

卸载

删除插件

rm -rf ~/Library/Application\ Support/Developer/Shared/Xcode/Plug-ins/Alcatraz.xcplugin

删除Alcatraz

rm -rf ~/Library/Application\ Support/Alcatraz

备注

安装完Alcatraz需重启Xcode。
如果在Xcode -> Window下，仍然找不到 Package Manager，请在Xcode选项中点QuitCode关闭。再次打开Xcode.

猫神的注释神器 —— VVDocumenter-Xcode

说明

开发时写注释是很重要的，经常写注释说明你是个负责任的工程师。在方法上一行，点击三下 ///便可自动根据方法参数生成注释。值得一提的是，它跟appleDoc简直是绝配,appledoc所生成的注释API文档与苹果类库的API文档保持一致.

图片提示神器 —— KSImageNamed-Xcode

说明

当我们输入[UIImage imageNamed:]自动弹出提示图片，选择图片回车就可以。省去了去图片库找图片复制文件名，再回到开发界面的过程。

三方库管理神器 —— cocoapods

说明

简单的命令行就可以查找，添加，更新，删除三方依赖库。主要是不用配置。（用了几个项目，但给同事导源码，他们总是打不开或是报错，慢慢的简单的项目就不用他了）

代码自动补齐 —— FuzzyAutocompletePlugin

说明

个人觉得不咋好用，尽管很多人推荐。装了它，打一个字母出现好多提示，很不习惯。

Json格式化输出为模型的属性 —— ESJsonFormat-Xcode

说明

以前用JsonExport自动生成model层文件。后来接触了MJExtension,就不用JsonExport了。这个插件刚好配合MJExtension使用。
