  • 开发工具：Xcode 6.3 + iOS8.0
  • 项目职责：独立开发，参与软件的功能设计和具体模块的代码实现。

2.K.RUN

  • 软件环境：Mac OS X 10.10.2 + iOS模拟器 + 真机
  • 硬件环境：iPhone6 + MacBook Pro
  • 开发工具：Xcode 6.3 + iOS8.0
  • 项目职责：独立开发，参与软件的功能设计和具体模块的代码实现。
  • 项目描述：这个项目是给智能硬件智能车轮开发的一款手机操作端，将硬件安装到车轮，在手机端可以将app酷图商城里的gif酷图，或自定义的文字，或自己拍摄的照片通过蓝牙传给智能车轮，行驶的轮子像一个屏幕，将其展示出来。
  • 技术要点：图片取点切图，文字绘图，将数据转换成16进制传给硬件。AFNetworking网络请求，FMDB本地存储，SDWebImage图片缓存，MJRefresh下拉刷新，DroppableView部分拖拽效果，CoreBluetooth封装BluetoothHelper。友盟的推送，统计，自动更新和分享。
  • APP官网：http://www.k21.me/

3.酷镜

  • 软件环境：Mac OS X 10.10.1 + iOS模拟器 + 真机
  • 硬件环境：iPhone6 + MacBook Pro
  • 开发工具：Xcode 6.0 + iOS7.0
  • 项目职责：二人项目。项目主程，完成绝大部分模块开发。调试Bug，版本迭代。
  • 项目描述：这是一款用以提升个人生活绩效的软件应用，完全针对移动设备设计。设计简洁但是功能强大。用户可以利用本软件记录个人财务、健康、心态等领域的数据信息。通过个人绩效（Personal KPI)的分析系统。提供用户的实时生活绩效指标查询，并为用户提供最佳生活实践。
  • 技术要点：AFNetworking网络请求，Html5和Native混合开发，SQLite3大数据事务存储，XML数据解析，雷达图绘制，MD5加密，Base64加密，微信，新浪微博分享。
